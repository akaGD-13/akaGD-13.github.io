VaR/ES Risk Calculation Project

End-to-end toolkit for 5-day Value at Risk and Expected Shortfall across parametric, historical, and Monte Carlo methods, including documentation, design notes, and validation artifacts.

Overview

This project packages multiple VaR/ES engines (parametric, EWM, historical, Monte Carlo) for equity and option portfolios. It includes interactive entry points, automated plotting, and a full set of documents to explain the models, software design, and validation results.
